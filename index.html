GitHub Comment Helper - プライバシーポリシー

最終更新日: 2025年4月1日

1. はじめに

GitHub Comment Helper（以下「本拡張機能」）は、GitHubのコメント入力を支援するChrome拡張機能です。 開発者のM-Miyazakoは、ユーザーのプライバシーを尊重し、個人情報の保護に努めています。 このプライバシーポリシーでは、本拡張機能がどのように情報を収集、使用、保護するかについて説明します。

2. 収集する情報

本拡張機能は、以下の情報のみを収集します：

  • ユーザーが設定したオプション（画像幅、動画幅、テーブルヘッダー名など）

これらの情報はChrome Storageに保存され、ユーザーのブラウザ内にのみ保存されます。 当社のサーバーに送信されることはありません。

3. 情報の使用目的

収集した情報は、以下の目的でのみ使用されます：

  • 本拡張機能の機能提供（GitHubコメントの入力支援）
  • ユーザー設定の保存と適用

4. 第三者への情報提供

本拡張機能は、収集した情報を第三者に提供することはありません。 ただし、法的要請がある場合は、法律に従って情報を開示することがあります。

5. 情報の保護

本拡張機能は、ユーザーの情報をChrome Storageに保存することで、 ブラウザの提供するセキュリティ機能によって保護されています。 当社のサーバーには情報が送信されないため、サーバー側でのセキュリティリスクはありません。

6. ユーザーの権利

ユーザーは以下の権利を有しています：

  • 本拡張機能の設定画面からいつでも保存された設定を変更できる
  • 本拡張機能をアンインストールすることで、保存されたすべての情報を削除できる

7. Cookie（クッキー）の使用

本拡張機能は、独自のCookieを使用しません。 ただし、GitHubのウェブサイト自体はCookieを使用している場合があります。

8. プライバシーポリシーの変更

本プライバシーポリシーは、必要に応じて更新される場合があります。 重要な変更がある場合は、本拡張機能の更新情報として通知します。
